# 📝 Personal Notes

A secure full-stack Personal Notes application where users can create, read, update, and delete their personal notes after authentication. Each user's notes are private and accessible only after logging in.

## 🚀 Features

- 🔐 User Authentication (Register & Login)
- 🍪 JWT Authentication with Cookies
- 📝 Create Personal Notes
- 📖 Read Notes
- ✏️ Edit Existing Notes
- 🗑️ Delete Notes
- 👤 User-specific Notes
- 🔒 Protected Routes
- 📱 Responsive UI

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- Tailwind CSS
- EJS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcrypt
- Cookie Parser

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/personal-notes.git
```

### Navigate into Project

```bash
cd personal-notes
```

### Install Dependencies

```bash
npm install
```

### Start MongoDB

Make sure MongoDB is running locally.

### Run Project

```bash
node app.js
```

or

```bash
nodemon app.js
```

Visit:

```
http://localhost:3000
```
